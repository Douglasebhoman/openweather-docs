# OpenWeather API Documentation

Welcome to the OpenWeather API documentation. This guide will help you 
retrieve current weather data for any city in the world.

## What You Will Learn

- How to set up your OpenWeather account
- How to make your first API request
- How to read and understand the response
- How to handle errors

## Quick Start

Get the current weather for London:
```
GET https://api.openweathermap.org/data/2.5/weather?q=London&appid=YOUR_API_KEY&units=metric
```

Ready? Start with the [Authentication](authentication.md) guide.
